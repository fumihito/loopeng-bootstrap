## Purpose

Use this frame to surface assumptions that a person or team is making without noticing. It separates observed patterns from interpretations and looks for missing alternatives, repeated avoidance, and hidden commitments.

## When to use

- You are scanning notes, logs, or conversations for what was left unsaid
- The likely problem is an omission, not a settled claim
- You need to compare the current story with earlier decisions or retrospectives

## Workflow

1. List the observed patterns.
2. Trace the inference chain.
3. Test for repeated avoidance and absent alternatives before treating the omission as settled.
4. Compare the current notes with earlier notes or decisions.
5. Name the hidden commitment or fear if one is supported.
6. State what evidence would settle the blind spot.

## Output

- Observed patterns
- Candidate blind spots
- Supporting lenses
- Confidence
- Recommended next action
- Memory comparison
- Suggested next experiment or check

## Exit

Stop when the omission is named with enough evidence to act on, or state that the patterns survived inspection. If no blind spot survives, say so explicitly and give the settling condition for the remaining doubt.

## Adjacent frames

- Use `frame-wall` when the user's framing is the thing that needs pressure-testing rather than the hidden assumption alone.
- Use `frame-critical-review` when the hidden assumption has become a claim that can be checked against sources or evidence.
- Use `frame-inertia` when the main question is whether a judgment is being repeated by habit or authority.
- Use `frame-first-principles` when the problem still needs decomposition before any claim can be tested.
- Use `frame-axis` when the disagreement turns out to be two people using the same word for two different things, not a hidden assumption.

## Merged operational contract

Run two passes: extract observations without interpretation, then test each
candidate through these lenses:

- Argyris: stated values versus actual action and defensive reasoning
- Kegan: commitments and the change they prevent
- Schein: unspoken group assumptions and norms
- Language: repeated labels, omissions, euphemisms, and tense changes
- Memory: mismatch with earlier decisions, incidents, or established beliefs

For each candidate, separate evidence, inference, alternatives, confidence,
and the observation that would settle it. A lens is not proof. Report observed
patterns, candidate blind spots, supporting lenses, confidence, next action,
and a small check or experiment. If none survives, say so and retain the
settling condition.
