## Purpose

Use this frame when a decision sounds inherited rather than reasoned. It checks whether a judgment is driven by convention, authority, metric fixation, or simple inertia.

## When to use

- A rule or judgment feels borrowed from elsewhere
- You need to know whether the reason still applies
- The question is whether the choice survives independent justification

## Workflow

1. Classify the trap.
2. Ask the probe questions.
3. Test whether the judgment is context-specific or inherited.
4. State the self-authoring reason if it survives.
5. Name the next step if the judgment does not survive.
6. Say what evidence would settle the disagreement.

## Trap classification

- socialized-conformity
- goodhart-metric
- authority-defer
- inertia
- self-authoring

## Probe questions

### socialized-conformity

- Would you still make this decision if the norm did not exist?
- Can you explain the decision using reasons specific to this context?

### goodhart-metric

- If this metric is wrong, what would you observe instead?
- Is the optimized metric harming the real goal?

### authority-defer

- If the framework or authority did not exist, would the conclusion still hold?
- Does the original context that created the rule still apply here?

### inertia

- Is the original decision still valid under current conditions?
- Would you make the same choice if starting from scratch now?

## Output

- Trigger phrase
- Trap type
- Self-authoring probe
- Why the probe matters
- Next step

## Exit

End when the inherited judgment either survives with a context-specific reason or fails and needs rebuilding from scratch. State the settling condition either way.

## Adjacent frames

- Use `frame-wall` when the inherited judgment may be masking a deeper framing error.
- Use `frame-critical-review` when there is a claim to test rather than an inherited judgment to audit.
- Use `frame-first-principles` when the judgment needs to be rebuilt from scratch.
- Use `frame-blind-spot` when the concern is the hidden commitment that keeps the inherited judgment in place.

## Merged operational contract

Classify the trigger as socialized conformity, Goodhart metric fixation,
authority deference, inertia, or self-authoring. For each trap, ask whether
the decision would survive if the norm, metric, authority, or original context
were removed. Also ask what observation would reveal harm to the real goal.

Report the trigger phrase, trap type, probe, why the probe matters, the
self-authoring reason if the judgment survives, and the next step if it does
not. Coordinate with Cynefin when the inherited judgment also determines the
response domain; do not treat the framework itself as evidence.
