## Purpose

Use this frame to surface assumptions that a person or team is making without noticing.
It separates observed patterns from interpretations and looks for missing alternatives, repeated avoidance, and hidden commitments.

## Adjacent frames

- Use `frame-critical-review` when the hidden assumption has become a claim that can be checked against sources or evidence.
- Use `frame-inertia` when the main question is whether a judgment is being repeated by habit or authority.
- Use `frame-first-principles` when the problem still needs decomposition before any claim can be tested.

## Analysis lenses

- Pattern repetition
- Avoided alternatives
- Hidden assumptions
- Language cues
- Comparison against other notes, decisions, or retrospectives

## Deeper lenses

### Argyris

- Trace the inference chain instead of jumping from data to conclusion
- Compare stated theory with actual behavior

### Kegan

- Identify the stated improvement goal
- Identify the behavior that keeps undermining it
- Infer the hidden commitment or fear behind that behavior

### Schein

- Look for absent alternatives
- Look for unspoken obvious assumptions
- Note where the scope has been implicitly frozen

### Language cues

- Absolute language
- External attribution
- Lack of agency
- Avoidant cutoff
- Excessive self-minimization

### Memory comparison

- Compare current notes with earlier notes, decisions, or retrospectives
- Check whether the same issue repeats without actually changing

## Output structure

- Observed patterns
- Candidate blind spots
- Supporting lenses
- Confidence
- Recommended next action
- Memory comparison
- Suggested next experiment or check
