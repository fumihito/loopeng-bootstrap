# 自己適用 追補

この文書は、Phase 1 再発行指示の自己適用を進める際の補足規律を定める。

## 追加規律

- SA-WP2 完了後の作業は journal 化する。
- SA-WP3 完了後の作業は intent 宣言を義務化する。
- SA-WP4 完了時には expectedFailure の削除を gate PASS と同じコミットで扱う。
- SA-WP5 完了後は、報告データが次の run に引き継がれることを確認する。

## 運用上の注意

- 既存の gate 実装は変更しない。
- 失敗が出た場合は、まず証拠を集める。
- 仕様の誤りを見つけたら、実装側で取り繕わず、判断材料として報告する。
