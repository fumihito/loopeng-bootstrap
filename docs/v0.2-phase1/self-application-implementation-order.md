# 自己適用化 実装指示（順序再編版）

Phase 1 の自己適用は、次の順序で実施する。

## フェーズ

1. 提案 2 を実施し、空検証テストを是正する。
2. 提案 3 の step 0 として、関連指示文書をリポジトリへ取り込む。
3. SA-WP1 から SA-WP5 を、gate 先行方式で順に実施する。
4. 最後に Phase 1 完了報告を作成する。

## 共通ルール

- 各 WP は独立した検収可能単位として扱う。
- 完了宣言は gate の PASS 出力に限定する。
- `utils/audit_guard.py record` を併用する。
- `utils/phase1_gate.py` は変更禁止。

## 進行条件

- SA-WP2 以降では、journal あるいは intent の要件を満たす。
- それぞれの WP 完了コミットには、対応する gate PASS の証跡を含める。
